🐛 Mobile Debug Console

📱 Perfect for Mobile Debugging

This tool captures all console output and displays it in an overlay on your webpage. Great for debugging on mobile devices where you can't access the browser console!

Look for the small debug button (🐛) in the bottom-right corner!

🚀 How to Use

Simply include the MobileDebugConsole class in your project:

// Initialize the debug console const debugConsole = new MobileDebugConsole(); // That's it! All console output will now be captured // and displayed in the mobile-friendly overlay

The debug console will automatically:

  • ✅ Capture all console.log, console.info, console.warn, console.error, console.trace
  • ✅ Show line numbers and source files when available
  • ✅ Display objects and arrays in a readable format
  • ✅ Add a toggle button to show/hide the console
  • ✅ Work perfectly on mobile devices
  • ✅ Show collapsible stack traces for errors

⚙️ Configuration Options

// Initialize with custom options const debugConsole = new MobileDebugConsole({ startVisible: false, // Start hidden maxEntries: 200, // Keep more log entries buttonPosition: 'top-left' // Change button position });
